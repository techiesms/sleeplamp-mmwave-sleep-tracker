# Screenshots & photos go here

The main `README.md` already links to the filenames below. Drop matching images
into this folder (same names) and they appear automatically — no README edits
needed. Until then GitHub shows a "broken image" icon for the missing ones; that
is expected.

Capture the dashboard from a phone or browser at **http://sleeplamp.local** while
the lamp is running.

| Filename | What to capture |
|---|---|
| `dashboard-hero.png`   | Top of the dashboard — sleep-score ring, current state, vitals |
| `hypnogram.png`        | The Sleep Stages chart mid-session (tap a bar so a detail chip shows) |
| `night-report.png`     | The Night Report section after a finished session |
| `history.png`          | The History list with a session expanded (shows the delete button) |
| `lamp-controls.png`    | The Lamp section (presets + brightness) |
| `hardware.jpg`         | A photo of your assembled lamp / breadboard |
| `wiring.jpg`           | A close-up of the ESP32-S3 ↔ C1001 ↔ DHT11 ↔ RGB wiring |

Tip: keep images under ~1 MB each so the repo stays light. PNG for screenshots,
JPG for photos.
