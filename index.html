Installation Steps

  1. Connect the ESP32-S3 using a USB data cable.
  2. Close Arduino IDE, PlatformIO or Serial Monitor.
  3. Click Flash Firmware.
  4. Select the correct COM Port.
  5. Wait until flashing completes.
  6. Device will reboot automatically.

Important

  • Use a USB DATA cable.
  • Remove any application using the COM port.
  • Do not disconnect power while flashing.
  • Use the latest Chrome or Edge browser.
